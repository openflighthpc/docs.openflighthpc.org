    Import Flight Solo Image to OpenStack

    Prepare for Image Import

    CLI Prerequisites

    1. To set this up, you will need to install the OpenStack CLI
    2. As well as the CLI, you will need your OpenStack RC file

    Upload Image

    1. Download the Flight Solo OpenStack image here
    2. Upload the image 
      openstack image save --file /path/to/Flight_Solo_VERSION_generic-cloudinit.raw Flight_Solo_VERSION_generic-cloudinit
